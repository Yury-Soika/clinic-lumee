# Lumee Clinic Demo

Standalone Next.js demo site for a fictional aesthetic medicine clinic.

## Development

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Production

```bash
npm run build
npm start
```

For CloudLinux / NodeJS Selector hosting, use:

```bash
./deploy.sh
```

The deploy script intentionally does not package `.env`.
